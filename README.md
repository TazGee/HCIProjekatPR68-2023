# WPF .Net Projekat - Nikola Tasić PR68-2023
Tema projekta: **Sistem za upravljanje sadržajima**  
Predmet: **Inženjerstvo upotrebljivosti u infrastrukturnim sistemima**
## Cilj projekta
- Izrada WPF aplikacije koja će imati funkcionalnosti dodavanja, brisanja i izmene liste objekata jedne klase. Objekti se čuvaju u .xml bazi podataka. Korisnici se takođe čuvaju u .xml bazi podataka. 
Baza podataka je lokalna i validna samo lokalno na računaru.  
- Aplikacija omogućava registraciju 2 vrste korisnika, **Admin** i **Posetilac**. Posetilac može samo da posmatra informacije o objektima, dok Admin ima opciju da manipuliše objektima.
## Korišćene tehnologije i principi
- Visual Studio 22
- WPF i .net 8.0 za frontend i backend
- SOLID principi
